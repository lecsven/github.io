设置hexo deploy

发表于 2016-06-26

每次使用 hexo generate后再使用hexo deploy都得输入一次usename和password,
好麻烦，就想着能不能一次性解决。

用Google搜了下。找到两篇网页：

生成 SSH 公钥

利用SSH省去在hexo deploy输入密码

可是设置好了SSH公钥,想要把传输改为ssh协议却总是报错

首先，很难确定 .deploy/ 文件的位置

使用 origin 也不行，总是报 fatal: No such remote ‘origin’

无从下手，希望能快点解决吧。

更新：

让Github 记住密码的方法如下：

1.查看 Git 和 osxkeychain helper 是否已经安装好

2.让Git使用osxkeychain helper，通过全局credential.helper

之后使用Github如果需要输入账号密码，系统会自动保存，以后就不用每次都输入了。

搞定！

参考文献：Caching your GitHub password in Git
